Failed to Install Redhat 7: Dracut-Initqueue Timeout

2023-03-15 Linux

找到你的U盘驱动器名称，例如: /dev/sdd1

重启后等到install页面时，按e进去启动介质编辑页面。
把：
inst.stage2=hd:LABEL=CentOS\x207\x20x86_64.check quiet
改为：
inst.stage2=hd:/dev/sdd1 quiet

最后按Ctrl+x保存重启就好了。
