Install Vim9 Supporting Python3

2022-07-07 Linux

In June 28th, 2020 the major version of Vim 9.0 was officialled released. You can download Vim 9.0 for appropriate platform at https://www.vim.org/download.php.

To download latest source code from Github by git clone https://github.com/vim/vim.git.

The installation procedure from source code is - (You’d have Python3 installed already in your Linux system.)

If error showing python3 can be be found occurs in the step of configure, make a symbolic link of python3 before doing configure as below.

The new installed Vim9 is located at /usr/local/bin/vim.

Check the Vim version infos by command vim --version. It shows +python and +python3 are configured correctly.
