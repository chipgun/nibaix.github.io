Update GLIBC 2.17 in RHEL6

2022-07-16 Linux

Contents

  1. From RPMs
  2. From Source Code

From RPMs

  1. Download RPMs
    http://copr-be.cloud.fedoraproject.org/results/mosquito/myrepo-el6/epel-6-x86_64/
  2. Install RPMs
    sudo rpm -Uvh glibc-2.17*.rpm

** You might encounter some errors like “error: Failed dependencies:…”. **
To fix it just add this parameter --force --nodeps at the end of the rpm command.
After finished up let’s check again.
strings /lib64/libc.so.6 | grep GLIBC
or
sudo yum list | grep -i glibc
